# Gestion de productos

Aplicacion ASP.NET CORE MVC con Entity Framework Core.

# Requisitos

- Net. SDK 11
- Visual Studio 2026
- Visual Studio Code
- Git

# Instalacion
- Clonacion del repositorio 

# Para correr en Visual Studio Code
- Abrir el proyecto /cd GestionPortal
- Abrir una terminal dentro del proyecto
- Ejecutar el comando dotnet restore
- Entrar en la subcarpeta /cd GestionPortal.web
- Ejecutar el comando dotnet run

# Para correr en visual Studio 2026
- Abrir el proyecto por Visual Studio
- Abrir una terminal
- Ejecutar el comando dotnet restore
- Iniciar depuracion
